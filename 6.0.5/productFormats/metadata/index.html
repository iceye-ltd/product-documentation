    Metadata Reference

    Metadata is the term used to describe all the ancillary information about an image that might be important to the user.

    This page captures all the metadata items that are present in ICEYE SLC and GRD Cloud Optimized GeoTIFF data products. The table indicates whether the metadata is available in the accompanying JSON metadata file, embeded as metadata in the GeoTIFF file or available in both. You can use the search bar at the top from any page and enter the metadata element you are interested in and it will find it on this page.
